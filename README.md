Everything you need to know about the CompTIA A+ 220-1201 (Core 1) exam


If you’re trying to break into IT, CompTIA A+ is a widely recognized entry-level certification—and a common first step into tech. There are no prerequisites, which makes it ideal for someone just getting their feet wet.

Exam Structure
• Two exams: 220-1201 (Core 1) and 220-1202 (Core 2)
• Up to 90 questions per exam
• 90 minutes per exam
• Multiple-choice + Performance-Based Questions (PBQs)
• Taken online or at a Pearson VUE testing center

What Core 1 Covers
• Mobile Devices
• Networking
• Hardware
• Virtualization & Cloud Computing
• Hardware & Network Troubleshooting

Cost
• ~$246 USD per exam
• Discounts available via CompTIA Store Club, student pricing (do not sleep on this), or employers

Career Paths A+ Helps Prepare You For
• Help Desk Technician
• Technical Support Specialist
• IT Support Technician
• Field Service Technician
• Desktop Support / Tier II Support

A+ won’t make you an expert—but it proves baseline IT competency, which is exactly what many entry-level roles are screening for.
